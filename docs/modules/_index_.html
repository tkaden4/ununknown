recursive

  • recursive<R, E, I>(body: function): Parser<R, E, I>
    • Defined in index.ts:89

    Due to javascript not being a "lazy" language, we have to embed recursive references to parsers in a thunk, like so:

    example
    interface Trie {
  value: string;
  children: Array<Trie>;
}

const trieParser: Parser<Trie> = recursive(() =>
  object.just({
    value: field.required(thing.is.string),
    children: field.required(array.of(trieParser))
  })
);
