recursive

  • recursive<R>(body: function): Validator<R>
    • Defined in index.ts:49

    Due to javascript not being a "lazy" language, we have to embed recursive references to validators in a thunk, like so:

    example
    interface RoseTree {
  value: string;
  children: Array<RoseTree>;
}

const roseTreeValidator: Validator<RoseTree> = recursive(() =>
  object.just({
    value: field.required(thing.is.string),
    children: field.required(array.of(roseTreeValidator))
  })
);
